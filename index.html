⚙ Setup Google Sheets

Dashboard ini membaca data langsung dari Google Sheets Anda. Ikuti langkah berikut untuk menghubungkan:

  1. Buat Google Sheets baru — gunakan template yang sudah disediakan bersama file ini (sheet: BudgetItems, Undangan)
  2. Klik File → Share → Publish to web
  3. Pilih tab BudgetItems → format CSV → klik Publish → copy URL
  4. Paste URL CSV BudgetItems di bawah
  5. Ulangi untuk tab Undangan
