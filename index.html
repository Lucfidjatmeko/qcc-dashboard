⚙ Setup Google Sheets

Hubungkan dashboard ke Google Sheets Anda melalui Apps Script. Ikuti langkah berikut:

  1. Buka Google Sheets baru → beri nama "QCC Dashboard FY26"
  2. Klik Extensions → Apps Script
  3. Hapus semua kode yang ada, paste isi file Code.gs yang sudah disediakan
  4. Klik Deploy → New Deployment
  5. Type: Web App · Execute as: Me · Access: Anyone
  6. Klik Deploy → Authorize → Copy URL
  7. Paste URL di bawah ini ↓
